# 🚀 Pokédex React

Aplicación moderna de Pokédex desarrollada en **React** con diseño responsive y animaciones fluidas. Consume la API pública de Pokémon (PokeAPI).

---

## ✨ Características

- 🎨 **Diseño Glassmorphism** con efectos de blur y transparencias
- 📱 **Totalmente Responsive** - funciona en móviles, tablets y desktop
- ⚡ **Animaciones suaves** con CSS
- 🔍 **Búsqueda inteligente** - busca por nombre de Pokémon o tipo
- 📊 **Estadísticas visuales** con barras de progreso animadas
- 🖼️ **Imágenes de alta calidad**
- ⚠️ **Manejo de errores** elegante

---

## 🚀 Cómo ejecutar

### 1. Instalar dependencias
```bash
npm install
```

### 2. Iniciar servidor de desarrollo
```bash
npm run dev
```

### 3. Abrir en el navegador
Vite te mostrará la URL (generalmente `http://localhost:5173`)

---

## 📱 Cómo usar

1. **Buscar un Pokémon**:
   - Escribe el nombre en inglés (ej: "pikachu", "charizard")
   - Presiona Enter o click en "Buscar"

2. **Buscar por tipo**:
   - Escribe un tipo (ej: "fire", "water", "electric")
   - Verás una grilla con 12 Pokémon de ese tipo
   - Click en cualquiera para ver sus detalles
